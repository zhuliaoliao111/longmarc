# 长征云迹 - 微信小程序

## 项目概述

<img src="pictures/小程序图标.png" alt="小程序图标" style="zoom:50%;" />

"长征云迹"是一款结合微信运动数据与长征历史教育的互动式微信小程序。通过将用户的日常运动步数转化为虚拟长征路程，让用户在健康运动的同时，沉浸式体验长征历史，了解长征中的重要事件和文物。

## 核心功能

### 1. 用户管理与数据存储（[login](pictures/login.jpg) & [user-home](pictures/user-home.jpg)& [user-setup](pictures/user-setup.jpg))

- 微信授权登录后经过用户允许后获取步数
- 个人虚拟长征进度与解锁章节记录

### 2. 与战士对话（[dialog](pictures/dialog.jpg) & [dialog_chat](pictures/dialog_chat.jpg))

+ 沉浸式跨时空互动体验
+ 用户可与名为李明远的红军战士交流，通过对话交流领悟长征精神

### 3. 文物讲长征([object](pictures/object.jpg)  & [object_*](pictures/object_.jpg) & [vr](pictures/vr.jpg))

- VR全面展示长征相关文物：八角帽、草鞋、防寒衣、藤饭碗、标语牌、银元
- 介绍文物来源并添加音频站在文物的视角讲述长征故事

### 4. 长征回忆录（[voice_diary](pictures/voice_diary.jpg) & [voice_*](pictures/voice_.jpg))

+ 收录《离开苏区上征途》，《周副主席到我团》，《八过赤水甩敌人》，《毛儿盖见朱总司令》，《决战山城堡再上新征途》五本日记
+ 支持字体选择，可调整字体大小、行间距和页边距、两种翻页模式
+ 支持语音朗读

### 5. 诗词绘长征（[challenge](pictures/challenge.jpg) & [ai-image](pictures/ai-image.jpg) & [ai-text](pictures/ai-text.jpg) & [poetry-cards](pictures/poetry-cards.jpg))

+ 以心看长征：展示经典长征诗词及详细注解，包含《七律·铁血军魂》《沁园春·长沙》等作品
+ 以字忆长征：用户输入关键词，大语言模型生成长征主题的诗词、散文或故事，支持古典、现代、豪迈三种风格
+ 以笔绘长征：大语言模型根据用户输入文本生成图像，重现长征场景

### 6. 重走长征路（[track1](pictures/track1.jpg) & [track2](pictures/track2.jpg) & [map](pictures/map.jpg))

- 用户可通过跑步获取长征虚拟里程，并记录运动总距离和轨迹图像
- 通过长征虚拟里程解锁各章节的长征故事

### 7. 长征景忆重燃([scene_*](pictures/scene.jpg))

- 复现八大经典红色场景：瑞金出发、湘江战役、遵义会议、四渡赤水、强渡大渡河、飞夺泸定桥、腊子口战役、会宁会师
- 每个场景中用户可以沉浸式参与长征决策，体会长征故事

## 技术架构

### 前端

- **微信小程序原生开发**：采用WXML、WXSS、JavaScript的原生开发模式
- **资源预加载与缓存优化**：针对长征场景图片、音频等大体积资源实现预加载机制，使用本地缓存策略减少网络请求，提升页面切换流畅度
- **响应式布局**：适配不同尺寸的移动设备屏幕，确保在各种机型上的良好显示效果
- **用户界面设计**：采用红色主题配色，结合长征元素的视觉设计语言

### 后端

- **微信云开发平台**：基于微信云开发技术栈
- **云函数处理业务逻辑**：部署7个核心云函数，处理用户认证、数据获取、资源上传等核心业务
- **云数据库存储用户数据**：采用数据库持久化存储用户运动数据、学习进度、互动记录等信息，支持数据查询和统计分析
- **云存储管理图片和音频资源**：集中管理长征场景图片、文物资料、音频讲解等多媒体资源，提供稳定的访问服务

### 核心工具类

#### 1. imageManager.js 图片管理工具类

+ 功能完善的图片管理系统

+ 主要负责小程序中所有图片资源的预加载、缓存管理、URL获取和上传功能

+ 通过优化的预加载策略提升用户体验

#### 2. audioManager.js 音频管理工具类

+ 封装了微信小程序的 InnerAudioContext API

+ 提供统一的音频控制接口，支持音频播放、暂停、停止等操作，并具有完善的事件监听机制

#### 3. tencentTTS.js 腾讯云语音合成工具

+ 腾讯云语音合成工具

+ 提供专业的文本转语音服务，支持音频播放

#### 4. crypto-js.js 加密工具库

+ 用于生成腾讯云混元文生图API的 TC3-HMAC-SHA256 签名，实现了 SHA256 哈希计算和 HMAC-SHA256 加密功能
+ 用于生成腾讯云语音合成 API 的签名，实现了规范的 TC3-HMAC-SHA256 签名算法
+ 用于与腾讯云混元大模型 API 交互时的签名生成

## 主要云函数说明

| API名称          | 使用位置                          | 功能说明                                      | 调用时机                                          |
| :--------------: | :-------------------------------: | :-------------------------------------------: | :-----------------------------------------------: |
| login            | app.js                            | 用户登录认证，接收微信登录code并返回会话信息   | 小程序启动时在login()方法中调用                   |
| getWeRunSteps    | map.js                            | 处理微信运动数据，获取用户步数信息            | 在loadWechatSteps()方法中调用，获取用户今日运动数据 |

## 项目特点   

#### 一、项目定位与创意核心

"长征云迹"是一款极具创新性的微信小程序，巧妙地将用户的日常运动步数与长征历史教育相结合，通过将真实运动步数转化为虚拟长征路程，实现了"运动即学习"的教育新模式。这种设计不仅鼓励用户积极参与体育锻炼，更让用户在健康生活的同时，沉浸式地体验长征历史，了解长征中的重要事件和文物故事，达到体育锻炼与爱国主义教育的双重目标。

#### 二、项目特色与创新亮点

##### 1. 教育模式创新
- 打破传统灌输式历史教育模式，开创"运动+教育"的创新学习方式
- 通过游戏化设计和即时反馈机制，大幅提升学习动力和参与度
- 将抽象的历史知识转化为可感知、可参与的互动体验

##### 2. 技术与历史的深度融合

- 充分利用微信小程序生态优势，降低用户使用门槛
- VR、AI等现代技术与红色文化内容的有机结合，为历史教育注入新活力
- 多媒体表达方式全方位还原长征历史

##### 3. 个性化学习体验
- 根据用户运动数据和学习进度，提供个性化的内容推送
- 多样化的互动模式满足不同用户的学习偏好
- 灵活的内容消费方式适应不同场景需求

##### 4. 完善的隐私与安全保障
- 严格遵循《中华人民共和国个人信息保护法》等相关法律法规
- 明确的隐私政策和用户协议，保障用户数据安全
- 采用加密技术保护用户信息，合理控制权限获取

#### 三、社会价值与意义

- 为红色文化传承提供了数字化、智能化的创新路径
- 将爱国主义教育融入日常生活，实现"润物细无声"的教育效果
- 激励广大用户特别是青少年群体参与体育锻炼，践行健康生活方式
- 为类似主题的教育类应用提供了可借鉴的技术架构和功能设计模式
